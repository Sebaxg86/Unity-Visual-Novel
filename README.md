# Entre tu Silencio y el Mío 🚀

Visual novel narrativa desarrollada en Unity 6, centrada en el primer capítulo de la historia de Jihuun, Seongsu y Jeongho.

## 🎮 Build jugable

Actualmente el proyecto no cuenta con una demo web pública. Su distribución se realiza mediante una build local de Windows.

## ⚙️ Tecnologías utilizadas

- Unity 6
- C#
- TextMesh Pro
- Unity UI (uGUI)
- Universal Render Pipeline 2D (URP)
- New Input System

## 📚 Descripción

Este proyecto fue desarrollado como un esfuerzo de reconstrucción, organización y cierre de una primera vertical slice jugable del capítulo 1. El trabajo actual incluye:

- ✅ Menú principal funcional con música de fondo
- ✅ Capítulo 1 jugable de inicio a fin
- ✅ Sistema de diálogo, retratos, pensamientos y señas
- ✅ Sistema de decisiones con impacto en confianza
- ✅ HUD de confianza, transiciones, SFX y cierre de capítulo
- ✅ Build local jugable para Windows

La documentación técnica y narrativa del proyecto se encuentra en [Docs/README.md](Docs/README.md).
